Desktop Application
## 팀명 및 팀원
* Red Light
* 나상혁 - 팀장, 기획, S/W 설계, S/W 개발
* 김건우 - 디자인, H/W 설계, H/W 개발, 기술조언
* 김혜령 - 디자인
* 윤석진 - S/W 개발

## 프로젝트 제목
* Red Light
* 행동 불능 시에도 사용 가능한 응급 호출 서비스

## 프로젝트 배경 혹은 목적
* 행동불능 상태에서도 위험을 알릴 방법이 필요
* 사용자의 조작 없이도 위험정보를 알릴 수 있는 제품
